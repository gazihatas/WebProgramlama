None (Görünmez)

Bir div'i görünmez yapabilirsiniz. Bu metotlarla genelde CSS menüler dediğimiz navigasyon araçları yapılır. Veya sayfa ortasında açılan reklamların kapatılması için x butonuna tıklanması örneğide verilebilir.

    Aşağıdaki elementlerde kullanılabilir.

  • div
  • h1-h6
  • p
  • form
  • footer
  • section

Örnekte: Butona tıklandığında ilgili fonksiyon tetikleniyor. Fonksiyon içinde de kutu id'li elementin display özelliğine none değeri atanarak görünmez yapıyor.

Kapat butonuna tıklarsanız bu kutu kapanacaktır.
